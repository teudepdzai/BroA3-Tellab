# Story Tactics Simulator

A digital simulation of the Story Tactics card deck system, built with React, Vite, and Tailwind CSS.

## Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or higher recommended)
- [VS Code](https://code.visualstudio.com/)

## How to Run Locally

1.  **Open the project in VS Code**
    -   Launch VS Code.
    -   Go to `File` > `Open Folder...` and select the folder containing this project.

2.  **Install Dependencies**
    -   Open the integrated terminal in VS Code (`Ctrl + \`` or `View` > `Terminal`).
    -   Run the following command to install all required packages:
        ```bash
        npm install
        ```

3.  **Start the Development Server**
    -   In the terminal, run:
        ```bash
        npm run dev
        ```

4.  **Open in Browser**
    -   The terminal will show a local URL (usually `http://localhost:5173` or `http://localhost:3000`).
    -   Ctrl+Click the link or open your browser and paste the URL to view the app.

## Features

-   **Interactive Card Grid**: Explore 8 categories of story tactics.
-   **Workspace**: Dedicated writing space for each card with prompts and auto-save.
-   **Custom Cards**: Create and save your own tactic cards.
-   **Timer**: Built-in stopwatch for writing sprints.
-   **Vietnamese Support**: Optimized fonts and interface for Vietnamese users.

## Tech Stack

-   React 19
-   Vite
-   Tailwind CSS 4
-   Framer Motion (animations)
-   Lucide React (icons)
